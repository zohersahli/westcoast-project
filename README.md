# Westcoast Education Platform 🎓

## Overview
A modern educational platform built for WestCoast Education using vanilla JavaScript and TypeScript. The platform provides course management, student registration, and booking functionality.

## Core Features
- Course browsing and booking system
- User authentication (students & admin)
- Admin panel for course management
- Responsive design for all devices
- Automated course cancellation system

## Tech Stack
- Frontend: Vanilla JavaScript, TypeScript, ES6 Modules
- Backend: JSON Server (REST API)
- Testing: Jest (TDD approach)

### Admin Features
- Add/Edit/Delete courses
- View student bookings
- Manage course schedules
- Monitor course registrations
- Auto-cancellation system for under-booked courses

## Quick Start
```bash
# Install dependencies
npm install

# Start JSON Server
npm run json-server

# Build and run application
npm run build && npm start

# Run tests
npm test
```
